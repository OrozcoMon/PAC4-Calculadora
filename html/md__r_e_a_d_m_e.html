Requirements

  • Windows 10/11
  • **.NET 8.0 Runtime**
  • Visual Studio 2022 (for development)

Features

Operator Precedence

  • Evaluates * and / before + and - (e.g., 5 + 3 * 2 = 11)

Basic Operations

  • Addition
  • Subtraction
  • Multiplication
  • Division

Additional Features

  • Clear (C button)
  • Error handling
  • Chained operations

Usage

  1. Basic operations:
    • Click number buttons (0-9)
    • Click operation buttons (+, -, *, /)
    • Press = for result
  2. Clear:
    • Press C to reset
  3. Chained operations:
    • Enter multiple operations (e.g., 5 + 3 * 2)
  4. Error handling:
    • Invalid inputs show "Error"
